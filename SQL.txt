1. Выведи список логинов курьеров с количеством их заказов в статусе «В доставке»

SELECT "Couriers".login, COUNT("Orders"."courierId") FROM "Couriers" INNER JOIN "Orders" ON "Couriers".id = "Orders"."courierId" WHERE "Orders"."inDelivery" = True GROUP BY "Couriers".login;

2. В ыведи все трекеры заказов и их статусы. 

SELECT track, CASE WHEN finished = true THEN 2 WHEN cancelled = true THEN -1 WHEN "inDelivery" = true THEN 1 ELSE 0 END FROM "Orders";
